
# HPLabelStandard

This module draws a label graphic for the unit's HP.

# User configuration

This module assumes that the graphics and palette are already in VRAM and PalRAM.

## Simple definitions

  * `HP_LABEL_X` and `HP_LABEL_Y`: The coordinates of the label in tiles
  * `HP_LABEL_BASE_TILE`: The VRAM tile index of the start of the label graphics
  * `HP_LABEL_PALETTE`: The PalRAM index of the label palette
  * `HP_LABEL_WIDTH`: The width of the label in tiles
